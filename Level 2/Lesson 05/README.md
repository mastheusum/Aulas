# Desafios da OBI pt 1

1. Faça um programa que mostre na tela o seu nome
2. Faça um programa que pergunte o nome do usuário e depois mostre na tela uma mensagem de bom dia com o nome digitado
3. Faça um programa que mostre os números de 0 à 100
4. Faça um programa que receba um número maior que 10 do usuário e mostre na tela os números de 0 até o número digitado
5. Faça um programa que mostre na tela a soma dos números de 0 à 100
   ```py
   somador = 0
   for i in range(101):
      somador = somador + i
   print(somador)
   ```
6. Faça um programa que receba um número maior que 50 do usuário e mostre na tela a soma dos números de 0 até o número digitado
7. Faça um programa que receba um número e se o número for maior que 6 mostre uma mensagem na tela, caso contrário mostre uma mensagem diferente
8. Faça um programa que receba duas notas do usuário, calcule e mostre na tela a média entre as notas digitadas
   ```py
   # sendo a e b as notas a média é calculada como
   media = ( a + b ) / 2
   ```
9. Faça um programa que peça para o usuário digitar um número, enquanto este número for diferente de zero o programa deve pedir um novo número 
10. Faça um programa que peça para o usuário digitar um número, e some este número aos números digitados anteriormente, enquanto este número for diferente de zero o programa deve pedir um novo número. Ao final deve se mostrar a soma dos números digitados
